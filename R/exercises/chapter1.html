R4DS Chapter 1 - Data Visualisation with ggplot2

Chapter 1 focuses on the ggplot2 package. You will learn how to use ggplot2 to create visualise a data set in a versatile and efficient manner.

ggplot2 is a package, itself contained within the tidyverse collection of packages. To use it we must first install the tidyverse package.

install.packages("tidyverse")

Once installed, we can then call on the package whenever we need using the following line.

library(tidyverse)

First Steps

1. Run ggplot(data=mpg). What do you see?

We see nothing. We have simply provided ggplot2 with a dataset, but haven't told the computer to do anything with it.

2. How many rows are in mtcars? How many columns?

Use dim(mtcars), which returns 32 11. ie mtcars has 32 rows and 11 columns.

3. What does the drv variable describe? Read the help for ?mpg to find out.

Using ?mpg, we are provided with a description of the drv column in the mpg dataset.

'drv - the type of drive train, where f = front-wheel drive, r = rear wheel drive, 4 = 4wd'

4. Make a scatterplot of hwy versus cyl 
          
          ggplot(data = mpg, mapping = aes(x = cyl, y = hwy)) + 
            geom_point()
5. What happens if you make a scatterplot of class versus drv? Why is this plot not useful?+ 
          
          ggplot(data = mpg, mapping = aes(x = drv, y = class)) + 
            geom_point()

By plotting two discrete variables against each other, especially those with such a small number of values, we can only draw limited conclusions from the graph. We can see which combinations of vehicle class and drivetrain were present in the sample of vehicles, and that's about it.
